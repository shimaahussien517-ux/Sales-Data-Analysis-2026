# 📊 Sales Data Analysis & Forecasting (2026)

Welcome to my advanced data portfolio! This repository contains a complete pipeline for processing, segmenting, and forecasting sales data using Python, Pandas, and Machine Learning.

3. **Sales Forecasting (ML):** Trained a Linear Regression model using Scikit-Learn to predict future sales trends.
